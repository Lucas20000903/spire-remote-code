현재 구현 vs ChatGPT/Claude 스타일 입력창

현재 구현
현재 chat-input.tsx
목표 스타일 옵션

A. ChatGPT 스타일

둥근 pill 형태 컨테이너, 하단에 아이콘 버튼들, 전송 버튼은 원형

특징

  • 둥근 모서리(24px radius)
  • textarea와 버튼이 컨테이너 안에 통합
  • 하단 툴바 (첨부, 모델선택 등)
  • 원형 전송 버튼

Claude can make mistakes. Please double-check responses.

B. Claude 스타일

둥근 박스 컨테이너, 하단 툴바, 브랜드 컬러 전송 버튼

특징

  • 중간 둥근 모서리(16px radius)
  • 통합된 컨테이너 (textarea + 툴바)
  • 하단 좌측 도구 아이콘들
  • 둥근 사각형 전송 버튼 (브랜드 컬러)

C. 미니멀 하이브리드

현재 디자인 시스템에 맞춘 깔끔한 모던 스타일, 불필요한 요소 제거

특징

  • 둥근 컨테이너(20px radius)
  • border-t 구분선 대신 떠있는 느낌
  • 미니멀한 전송 버튼만
  • 현재 색상 시스템과 자연스러운 조화
